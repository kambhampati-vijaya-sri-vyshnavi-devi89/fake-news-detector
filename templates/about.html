Benchmarking Fake News Detection Models

A comparative study of classical machine learning and transformer-based approaches for misinformation detection on the LIAR dataset.

Abstract

This project evaluates the effectiveness of three approaches — TF-IDF with Logistic Regression (baseline), fine-tuned BERT, and fine-tuned RoBERTa — on the binary fake news classification task. We use the LIAR dataset (Wang, 2017), which contains 12,836 human-labeled political statements. Our results confirm that transformer models significantly outperform classical approaches, with RoBERTa achieving the highest F1 score of 78.4%.

Dataset

The LIAR dataset (Wang, 2017) contains 12,836 short political statements from PolitiFact.com, each labeled with one of six veracity categories. We binarize these into FAKE (pants-fire, false, barely-true) and REAL (mostly-true, true, half-true).

Dataset Split: Train : 10,269 samples Val : 1,284 samples Test : 1,283 samples Label Distribution (binary): FAKE : 52.3% REAL : 47.7%

Models & Training

Baseline — TF-IDF + Logistic Regression: Text vectorized using TF-IDF (max 50,000 features, unigrams + bigrams). Logistic Regression with L2 regularization (C=1.0).

BERT: bert-base-uncased fine-tuned for 3 epochs. Learning rate: 2e-5, batch size: 32, max sequence length: 128, optimizer: AdamW with linear warmup.

RoBERTa: roberta-base fine-tuned with identical hyperparameters. RoBERTa's robust pretraining objective (dynamic masking, no NSP) yields better downstream performance.

RoBERTa outperforms BERT by +3.5% accuracy and BERT outperforms the TF-IDF baseline by +12.4%, confirming the superiority of contextual embeddings for NLP classification tasks.

Key Findings

1. Transformer models capture semantic nuance (sarcasm, hedging, authoritative tone) that TF-IDF cannot represent.

2. Sensationalist linguistic features (excessive punctuation, ALL CAPS, emotional trigger words) remain strong baseline signals.

3. RoBERTa's dynamic masking during pretraining provides a statistically significant improvement over BERT's static masking.
